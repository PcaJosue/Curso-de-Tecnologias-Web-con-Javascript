Spread Operator

En ES5, podemos encontrar el máximo valor en un array utilizando el método Math.max, así:
Math.max.apply(null, [-1, 100, 9001, -32]); // 9001


En ES6, podemos utilizar el operador spread para pasar un array de valores para ser usado como parámetro en una función:
Math.max(...[-1, 100, 9001, -32]); // 9001


Podemos concatenar elementos de un array fácilmente con esta sintaxis intuitiva:
let cities = ['San Francisco', 'Los Angeles'];
let places = ['Miami', ...cities, 'Chicago']; // ['Miami', 'San Francisco', 'Los Angeles', 'Chicago']
