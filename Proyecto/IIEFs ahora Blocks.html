Problemas Con Bloques

IIEFs vs Blocks

Un uso común de las expresiones de función inmediatamente invocadas es incluir valores dentro de su ámbito de aplicación.
(function () {
    var food = 'Meow Mix';}());
console.log(food); // Reference Error
En ES6, ahora tenemos la capacidad de crear ámbitos basados en bloques y por lo tanto no están limitados puramente a ámbito basado en funciones.
{let food = 'Meow Mix';};
console.log(food); // Reference Error
