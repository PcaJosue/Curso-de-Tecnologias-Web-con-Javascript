Modules

Export

Antes de ES6, usábamos libreriías como Browserify para crear módulos en el lado del cliente, y requerirlas em Node.js. Con ES6, podemos directamente usar módulos de todos los tipos (AMD y CommonJS).
Exportar en CommonJS:
module.exports = 1;
module.exports = { foo: 'bar' };
module.exports = ['foo', 'bar'];
module.exports = function bar () {};
Exportar en ES6:
Con ES6, tenemos varios tipos de exportación. Podemos realizar Exportaciones Nombradas:
 export let name = 'David';
export let age  = 25;
Así como exportar una lista de objetos:
 function sumTwo(a, b) {
    return a + b;
}

function sumThree(a, b, c) {
    return a + b + c;
}

export { sumTwo, sumThree };
También podemos exportar funciones, objetos y valores (etc.) simplemente usando la palabra clave export:
 export function sumTwo(a, b) {
    return a + b;
}

export function sumThree(a, b, c) {
    return a + b + c;
}


Import

ES6 nos provee varias formas de importar. Podemos importar un archivo entero:
import 'underscore';
Similar a Python, tenemos importaciones nombradas:
import { sumTwo, sumThree } from 'math/addition';
Además, podemos importar todas las cosas 
import * as util from 'math/addition';
Por uúltimo, podemos importar una lista de valores desde un moódulo: 
import * as additionUtil from 'math/addition';
const { sumTwo, sumThree } = additionUtil;
