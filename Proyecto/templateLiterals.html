Template Literals

Usando Template Literals, ahora podemos construir cadenas que tienen caracteres especiales en ellas sin necesidad de escaparlas explícitamente.
 var text = "This string contains \"double quotes\" which are escaped.";
let text = `This string contains "double quotes" which don't need to be escaped anymore.`;
Los literales de plantilla también admiten interpolación, lo que hace que la tarea de concatenar cadenas y valores:
 var name = 'Tiger';
var age = 13;
console.log('My cat is named ' + name + ' and is ' + age + ' years old.');
Mas Simpe:
 const name = 'Tiger';
const age = 13;
console.log(`My cat is named ${name} and is ${age} years old.`);
Para concatenar varias lineas antes:
var text = (
    'cat\n' +
    'dog\n' +
    'nickelodeon');
Ahora:
let text = ( `cat
dog
nickelodeon`);
Los literales de plantilla también pueden aceptar expresiones:
let today = new Date();
let text = `The time and date is ${today.toLocaleString()}`;
